This repository contains labs for the HSE Java course.
Labs:
1. Complex number matrix calculator
2. Frequency dictionary: counting the number of letters in a text file
3. Working with strings: converting full name to initials, year of birth to age, and sex determination
4. Working with the API of two websites to get data about users and companies and parsing this data
5. Elevator control manager using multithreading and a graphical interface
